How to set up livestreaming:

1. Visit twitch.com and create an account.

2. Download the mobile twitch app for iphone or android (or stream from computer, but mobile is easier).

3a. For mobile: Login on the app -> go to your profile -> click "Go Live!"

3b. For Desktop: Go to your twitch dashboard and go live using broadcast software specified on the website

4. Enter in the name of your channel above and create your classroom.
